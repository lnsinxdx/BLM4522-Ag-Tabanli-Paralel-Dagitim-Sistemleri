# Proje 2 — Veritabanı Yedekleme ve Felaketten Kurtarma Planı

AdventureWorks2022 üzerinde yedekleme stratejileri ve felaketten kurtarma
senaryoları. Tam/fark/log yedekleri, point-in-time restore, tail-log yedeği
ile felaketten kurtarma ve HA/DR yöntemlerinin karşılaştırması.

## Ortam
- SQL Server (MSSQL) — AdventureWorks2022
- Windows VM (Parallels / Mac)

## Fazlar
1. Setup ve recovery model ayarı
2. Full / differential / log yedekleri
3. Point-in-time restore
4. Felaketten kurtarma + tail-log
5. Yeni veritabanına restore + HA/DR karşılaştırması
6. Test ve rapor
